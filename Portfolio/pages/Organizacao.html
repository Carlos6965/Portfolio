Dispositivos de Entrada e Saída & Barramentos

Dispositivos de Entrada e Saída são componentes que permitem a comunicação entre o usuário e o computador, já os barramentos são vias de comunicação que conectam os componentes internos do computador.

Dispositivos de Entrada (Input):

  • Teclado, mouse, scanner, microfone, câmera
  • Convertem ações do usuário em dados digitais

Dispositivos de Saída (Output):

  • Vias de comunicação entre componentes do computador
  • Apresentam informações processadas ao usuário

Barramentos:

  • Vias de comunicação entre componentes do computador
  • Barramento de Dados: Transporta informações (bidirecional)
  • Barramento de Endereços: Especifica localizações (unidirecional)
  • Barramento de Controle: Gerencia operações (sinais de clock, leitura/escrita)

Teoria dos Conjuntos

Teoria dos Conjuntos é um ramo da matemática que estuda conjuntos, que são coleções de objetos bem definidos, chamados elementos.

Ela fornece a base para grande parte da matemática moderna. Os principais conceitos:

  • Conjunto: coleção de elementos (ex: {1, 2, 3}).
  • Pertinência: um elemento pertence ou não a um conjunto.
  • Subconjunto: todo elemento de um conjunto A está em B.
  • União: reúne todos os elementos de dois conjuntos.
  • Interseção: elementos comuns a dois conjuntos.
  • Diferença: elementos de um conjunto que não estão no outro.
  • Conjunto vazio: conjunto sem elementos (∅).

A teoria dos conjuntos é fundamental para lógica, álgebra, análise e outras áreas da matemática.

Organização de Computadores & Sistemas Operacionais

Organização de Computadores é o estudo da estrutura e funcionamento dos componentes internos do computador, como processador, memória, dispositivos de entrada/saída e barramentos. Já Sistemas Operacionais são softwares que gerenciam os recursos do computador, permitindo a comunicação entre hardware e usuário.

Memória RAM:

  • Volátil, armazenamento temporário
  • Tipos: DRAM (mais comum), SRAM (mais rápida)
  • Organização em células endereçáveis

Memória ROM:

  • Não-volátil, contém firmware
  • Tipos: PROM, EPROM, EEPROM, Flash

Hierarquia de Memória:

  • Registradores (mais rápido, menor capacidade)
  • Cache (L1, L2, L3)
  • Memória Principal (RAM)
  • Armazenamento Secundário (HDD/SSD)
  • Armazenamento Terciário (fitas, cloud)

Esses temas são fundamentais para entender como computadores funcionam e como os recursos são gerenciados.

Unidade Aritmética e Lógica (ULA)

Unidade Aritmética e Lógica (ULA) é o componente do processador responsável por executar operações matemáticas (adição, subtração, multiplicação, divisão) e lógicas (AND, OR, NOT, XOR).

Componente central da CPU

  • Operações aritméticas: +, -, ×, ÷
  • Lógicas: AND, OR, NOT, XOR
  • Deslocamento: Shift, Rotate
  • Trabalha em conjunto com registradores e UC

Pipeline

Pipeline é uma técnica usada nos processadores para aumentar o desempenho, dividindo a execução de instruções em várias etapas sequenciais (estágios), como busca, decodificação, execução e escrita do resultado.

Técnica de processamento paralelo

Estágios típicos:

  • Fetch (busca)
  • Decode (decodificação)
  • Execute (execução)
  • Memory (acesso à memória)
  • Writeback (escrita)

Hazards: Estrutural, de Dados, de Controle

O pipeline é fundamental para alcançar altas velocidades em CPUs modernas.

Organização de Processadores

Organização de Processadores refere-se à forma como os processadores são estruturados internamente para executar instruções de maneira eficiente.

Von Neumann:

  • Memória unificada para dados e instruções
  • Gargalo no barramento

Harvard:

  • Memórias separadas
  • Maior paralelismo

Processamento Paralelo:

  • Multicore: Vários núcleos no mesmo chip
  • Hyperthreading: Núcleos lógicos
  • GPGPU: Processamento gráfico para fins gerais

Tópicos Adicionais:

  • Clock da CPU (frequência de operação)
  • Arquiteturas RISC vs CISC
  • Co-processadores (ex: FPU)
