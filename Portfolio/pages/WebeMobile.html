Técnicas Avançadas de Programação Web e Mobile

Matéria ministrada por: Marcelo Tadeu Boer

DOM (Document Object Model)

Objetivos:

  • Interface entre a página web e linguagens de programação (como JavaScript).
  • Representa a estrutura da página como uma árvore hierárquica (elementos pais, filhos e netos).
  • Facilita a manipulação dinâmica da página via JavaScript.

React JS

  • Biblioteca JavaScript para criar interfaces interativas.
  • Componentização: Uso de componentes reutilizáveis para construir UIs complexas.
  • Virtual DOM: Otimiza performance ao atualizar apenas partes modificadas da página.
  • JSX: Sintaxe que permite escrever HTML dentro do JavaScript.

React Router (Navegação em React)

Evita recarregar a página ao navegar entre rotas (Single Page Application - SPA).

Configuração básica (Exemplo: BlogTravel):

  • Estilização do Navbar (CSS Modules).
  • Links de navegação (ex: Home, About).
  • Uso de Google Fonts para tipografia.

Expansão do Projeto:

  • Criação de novas páginas (Login, Register).
  • Validação de formulários (ex: confirmação de senha).
  • Integração com backend (ex: Firebase) para registro de usuários.
