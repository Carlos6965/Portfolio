Ementa

Aborda conceitos fundamentais de Programação Orientada a Objetos (POO) e desenvolvimento de software

  • POO: Classes, objetos, encapsulamento, herança, polimorfismo, métodos, classes abstratas, sobrecarga de métodos, herança múltipla, modificadores de acesso, construtores.
  • Tópicos Avançados: Manipulação de exceções, arquitetura em camadas, interfaces gráficas, padrões de projeto (GoF, MVC, MVVM, MVP), Test-Driven Development (TDD), controle de versionamento.

Crud 1-N .NET com PostgreSQL

Implementar um CRUD (Create, Read, Update, Delete) para relacionamento 1-N (um fornecedor pode ter vários produtos) usando ASP.NET Core MVC e PostgreSQL.

Configuração Inicial:

  • Criação do projeto MVC no Visual Studio.
  • PInstalação de pacotes NuGet (Npgsql.EntityFrameworkCore.PostgreSQL, Microsoft.EntityFrameworkCore.Tools).
  • Configuração da conexão com PostgreSQL no appsettings.json.

Modelagem de Dados:

  • Classes Fornecedor e Produto com relacionamento 1-N.
  • Criação do DbContext (ApplicationDbContext) para acesso ao banco de dados.

Migrações e Banco de Dados:

  • Execução de comandos para criar migrações (Add-Migration Inicial) e atualizar o banco (Update-Database).

Scaffolding:

  • Geração automática de controllers e views para Fornecedor e Produto usando Entity Framework.
